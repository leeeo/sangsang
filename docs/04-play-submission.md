## 2. 스토어 등록정보 (Store listing)

**짧은 설명 (80자 이내)**
```
축의금·조의금 기록하면 상대방별 주고받은 금액 자동 정리. 회원가입 없는 경조사비 가계부
```

**자세한 설명**
```
상부상조는 경조사비(축의금·조의금·돌잔치·집들이 등)를 간편하게 기록하고,
상대방별로 주고받은 금액을 자동으로 정리해 주는 한국형 경조사비 가계부입니다.

■ 이런 분께 좋아요
· 축의금/조의금을 얼마나 주고받았는지 기억이 안 날 때
· 경조사 답례를 준비하며 지난 기록을 확인하고 싶을 때
· 엑셀 대신 폰에서 간단하게 관리하고 싶을 때

■ 주요 기능
· 거래 기록: 금액·날짜·상대방·경조사 유형·메모까지 한 번에
· 관계별 자동 집계: 상대방별 주고받은 총액과 잔액을 자동 계산
· 분석: 월별 수입/지출 요약, 최근 추이, 카테고리별 비율
· 기본 카테고리 15종 내장 (결혼 축의금, 장례 조의금, 돌잔치 등)

■ 가볍고 안전하게
· 회원가입 없음 — 이름만 입력하면 바로 시작
· 모든 기록은 내 기기에만 저장 (서버 전송 없음)
· 무료 (광고 포함)

※ 현재 버전은 기기 내 저장 방식이라 앱 삭제 시 기록이 함께 삭제됩니다.
   백업/내보내기 기능은 업데이트로 준비 중입니다.
```

**그래픽 애셋 (필수)**
- [ ] 앱 아이콘 512×512 PNG — ⚠️ 준비 필요 (앱 런처 아이콘도 현재 Flutter 기본 → 교체 필요)
- [ ] 피처 그래픽 1024×500 PNG — ⚠️ 준비 필요
- [ ] 스크린샷 (휴대전화 세로) 2장 이상 — 실기기 QA 때 캡처 권장: 홈 / 거래 등록 / 관계 / 분석

## 3. 데이터 보안 양식 (Data safety) — 심사 핵심 ⚠️

> 원칙: **앱 기록은 기기에만 저장(수집 아님)**, 단 **광고(AdMob) SDK가 광고 ID를 수집**한다.

| 질문 | 답 |
|---|---|
| 데이터를 수집하거나 공유하나요? | **예** |
| 수집 항목 | **기기 또는 기타 ID** (광고 ID) — 그 외 항목 전부 "수집 안 함" |
| 재무 정보(거래 내역 등) | **수집 안 함** (기기에만 저장, 전송되지 않음) |
| 개인 정보(이름·이메일 등) | **수집 안 함** (프로필 이름은 기기에만 저장) |
| 위치 | **수집 안 함** (광고의 IP 기반 대략적 위치는 "수집"에 해당 안 함 — AdMob 가이드 기준) |
| [기기 ID] 수집 목적 | 광고 또는 마케팅 |
| [기기 ID] 공유 여부 | 제3자(Google/광고 파트너)와 공유됨 |
| [기기 ID] 필수/선택 | 필수 (앱 기능상 광고 게재에 필요) |
| 전송 중 암호화 | 예 |
| 삭제 요청 방법 제공 | 예 — 앱 삭제 시 전체 데이터 삭제 + 문의 이메일 |
| 개인정보처리방침 URL | `https://leeeo.github.io/sangsang/privacy.html` |

## 4. 광고 / 콘텐츠 등급 / 타겟층

| 항목 | 답 |
|---|---|
| 앱에 광고가 포함되나요? | **예** (AdMob 배너/전면) |
| 콘텐츠 등급 설문 | 카테고리: 유틸리티/생산성 등 → 폭력/성적/도박/약물 전부 "아니요" → **전체이용가** 예상 |
| 타겟층(Target audience) | **만 18세 이상** 권장 (가계부 특성 + 아동 정책 리스크 제거. 처리방침도 "14세 미만 대상 아님" 명시됨) |
| 뉴스 앱인가요? | 아니요 |
| 코로나19 앱인가요? | 아니요 |
| 데이터 보안 양식과 광고 ID | AndroidManifest에 AD_ID 권한은 google_mobile_ads가 자동 포함 — "광고 ID 사용" 질문에 **예** |

## 5. 제출 순서 (요약)

1. 내부 테스트 트랙에 서명된 AAB 업로드 ([docs/03](03-android-release.md))
2. 테스터(본인 계정) 등록 → 실기기 설치 확인
3. 위 2~4장 양식 작성 완료
4. ⚠️ **신규 개인 계정 요건**: 프로덕션 공개 전 비공개 테스트(테스터 12명·14일) 요구될 수 있음 — 콘솔 안내에 따를 것
5. 프로덕션 제출 → 심사 (수일)
6. 게시 후: [index.html](../website/index.html)의 Play 스토어 링크 TODO 교체 + `website/app-ads.txt` 확인
